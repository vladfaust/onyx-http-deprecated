    module Onyx::HTTP::Rescuer(T)

    Overview

    Rescues T. Firstly calls #handle to handle the error (e.g. log it). Then if #next_handler is present, calls #before_next_handler and then the next handler itself. Otherwise calls #fallback.

    See Rescuers::Standard and Rescuers::Silent.
