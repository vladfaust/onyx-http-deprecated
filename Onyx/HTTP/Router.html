    class Onyx::HTTP::Router

    • Onyx::HTTP::Router
    • Reference
    • Object

    Overview

    Routes a request's path, then updates extracted path params to ::HTTP::Request#path_params, executes the matching proc and calls the next handler if it's present.

    Raises RouteNotFoundError if no route is found for the URL path, so it should be with a Rescuer in the stack.

    router = Onyx::HTTP::Router.new do
  get "/" do |env|
    env.response << "Hello world!"
  end

  ws "/" do |socket, env|
    socket.on_message do |message|
      # ...
    end
  end
end

router_rescuer = Onyx::HTTP::Rescuers::RouteNotFound.new

server = Onyx::HTTP::Server.new([router_rescuer, router])
